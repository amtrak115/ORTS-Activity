MSTS Six 52' Mill Gondolas in both loaded and empty versions for ATW 700234, CEFX 33534, CP 334327, CSXT 484053, DME 80081 and NOKL 320370.  Model and original textures and freight anim loads by Robert Murphy. Coupler and wheel textures by Ted Curphey. Bob Boudin's version 4.9 physics.  Repainted by Mark Giltz.  Photos provided by and used with the permission of Bryce Denny, Keith Belk and Paul Graf.

email: markuprailfan@aol.com

JANUARY 2009

Installation:

Unzip to your Trainset folder; a folder named "GONPACK1" will be created and the files added to it.  Use the activity editor or ConBuilder to add the carS to the Sim.
