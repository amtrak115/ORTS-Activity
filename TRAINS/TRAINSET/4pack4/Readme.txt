MSTS four 40' Funnel Flow tank cars FPAX 931080, FPAX 931106, FPAX 931142 and UTLX 11534 in both loaded and empty versions.  Model and original textures by Robert Murphy.  Physics by Bob Boudin.  Coupler and wheel textures by Ted Curphey. Repainted with permission from Robert Murphy by Mark Giltz.

Installation:

Unzip to your Trainset folder, a folder named "4PACK4" will be created
and the files added to it. Use the activity editor or ConBuilder
to add the car to the Sim.

See the original Readme file regarding the original author's rights and desires regarding these cars.
