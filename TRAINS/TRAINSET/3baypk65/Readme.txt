MSTS Eight PS2 3 bay hoppers for ATSF 315671, BN 468584, IFRX 309909, PTLX 1428, PTLX 34416, PTLX 34781, TLCX 30895 and UELX 60130 in both loaded and empty versions.  New shape file by David Rowe. New textures by Bernard Grant. New FF-Hardware.ace by Jeff Auberpine. Improved physics by Bob Boudoin.  Repainted by Mark Giltz.   Photos provided by and used with the permission of Ian Clasper, Erik Edmonds, Aaron Florin, Ron Hawkins and Mike Rujak.

Installation:

Unzip to your Trainset folder, a folder named "3BAYPK65" will be created
and the files added to it. Use the activity editor or ConBuilder
to add the car to the Sim.  Cars will appear as "HOPPER ATSF315671 Loaded", 
"HOPPER ATSF315671 Empty", etc.
